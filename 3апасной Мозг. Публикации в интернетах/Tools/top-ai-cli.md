# Топ AI CLI для повседневной работы

CLI-утилиты с ИИ полезны не тогда, когда впечатляют, а когда снимают рутину. В этом списке только то, что реально ускоряет повседневную работу.

## На что смотреть

- скорость ответа;
- предсказуемость команд;
- нормальная работа с файлами;
- возможность встроить в свою связку.

## Практичный набор

- Codex для инженерных задач и быстрого изменения проекта;
- `ripgrep` для поиска по коду и заметкам;
- `fd` для навигации по дереву файлов;
- `jq` для фильтрации JSON;
- `bat` для чтения файлов без боли.
