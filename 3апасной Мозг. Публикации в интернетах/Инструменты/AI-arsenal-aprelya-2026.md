# AI-арсенал апреля — 70+ нейросетей и инструментов

Полный разбор апрельского блицобзора канала **Нейроразведка** (Дмитрий). Все инструменты сгруппированы по категориям. Тип указан в скобках.

---

## Агрегаторы и поиск аналогов

1. **Pae** (сервис) — агрегатор 500+ нейросетей. Учёба, бизнес, маркетинг, кодинг, генерация контента. Запуск агентов в один клик. Браузер, ПК, Android, iOS. Бесплатно.

2. **Open Apps** (сервис) — подбор бесплатных open-source аналогов платным нейросетям (Notion, ChatGPT, Cursor и др.). Без регистрации.

3. **Toolool LFG** (сервис) — каталог онлайн-инструментов для бизнеса, творчества, маркетинга, разработки, дизайна. Удобная структура по категориям.

---

## Генерация изображений

4. **Z Image Turbo** (Alibaba, сервис + GitHub) — генератор без цензуры. Максимальный реализм, фотореалистичные результаты за секунды. Онлайн + локально.

5. **Flux 2** (GitHub) — генератор с загрузкой до 10 референсов. Улучшенная детализация, физика, оптика. До 4 Мп.

6. **Nano Banana** (Hugging Face) — мощный генератор изображений. Ранее платный, теперь бесплатно на HF. Визуалы, баннеры, креативы для бизнеса.

7. **Сборник промтов для Nano Banana** (GitHub) — сотни готовых запросов по категориям с примерами.

8. **Нестандартные кейсы для Nano Banana** (GitHub) — промты для комиксов, манги, Minecraft, виртуального макияжа.

9. **Upscale Prompt для Nano Banana** (промт) — апскейл изображений без потери деталей с сохранением текстуры.

10. **700+ промтов для ChatGPT Images 2.0** (подборка) — шаблоны, стили, реальные кейсы по категориям.

## Генерация видео

11. **Veo 3.1** (Google, сервис) — продвинутый генератор. Высокая детализация, свет, тени, понимает русский. До 10 генераций бесплатно в месяц.

12. **Neo** (GitHub) — китайский видеогенератор, аналог Nano Banana. По тексту, фото, референсу. Смена стилей, удаление водяных знаков.

13. **Cinema Studio** (сервис) — AI-киностудия. Полноценный продакшн по одному промту: режиссёр, сценарист, оператор. Для клипов, рекламы, трейлеров.

---

## Видео: монтаж, нарезка, дубляж

14. **Vicecat** (сервис) — AI-нарезка длинных видео в шортсы для TikTok, YouTube Shorts. Авто-поиск цепляющих моментов, добавление музыки и субтитров.

15. **Linly Dubbing** (GitHub) — дубляж и перевод видео с клонированием голоса и синхронизацией губ. 100+ языков, включая русский.

16. **Videolina** (GitHub) — перевод видео с сохранением оригинального тембра голоса. 8 языков, авто-субтитры, импорт с YouTube.

17. **Watermark Remover** (сервис) — удаление водяных знаков с фото и видео без потери качества. Высокая приватность.

## Голос и клонирование

18. **Qwen 3 TTS + 4 модели** (GitHub) — сборник из 5 топовых TTS-моделей. 10 языков, 49 голосов (пол, возраст, акцент, тембр). Задержка 97 мс. Всё локально.

19. **VoiceBX** (GitHub) — локальный голосовой редактор. Клон по короткому фрагменту, диалоги, подкасты, запись/расшифровка. Полностью офлайн.

20. **Omnivoice Studio** (GitHub) — транскрибация, перевод, клонирование от 3 сек, интеграция в видео, разделение вокала и музыки.

21. **Сервики** (сервис) — TTS: текст в речь с естественным тембром и интонациями. Отлично работает на русском.

## Музыка и звук

22. **AC Step** (GitHub) — генерация музыки. До 10 мин на трек, 1000 инструментов, сотни жанров, 50 языков. До 8 треков одновременно. Коммерческое использование разрешено.

23. **Audio Tools Collection** (GitHub) — сборник инструментов для работы со звуком: клонирование, генерация музыки, Docker.

24. **AOHON** (сервис) — очистка аудио до студийного уровня. Удаление шума, пауз, эканья. 2 часа бесплатно в месяц.

25. **Motion Graphics AI** (сервис) — генерация моушн-графики и анимации по тексту. Библиотека шаблонов, поддержка CSV и логотипов.

## Кодинг и AI-агенты

26. **OpenCode** (GitHub) — замена Claude Code. Проекты под ключ, 75+ провайдеров (OpenAI, Google, Mistral, локальные). Из терминала.

27. **Sitech** (Google, сервис) — вайбкодинг + дизайн интерфейсов. Gemini-агент: промт → интерфейс + код. Можно тестировать сразу.

28. **OpenCloud** (сервис + курс) — AI-агент с памятью. Анализирует интернет, собирает данные, формирует отчёты с источниками. Есть курс на русском.

29. **Сбер AI-агент** (сервис) — бесплатный агентный режим. Находит файлы, исправляет код, тестирует, собирает проекты.

30. **Pencel** (сервис) — генерация интерфейсов по промпту. Режим Roy — до 6 вариантов одновременно. Gemini, Opus, Sonet, OpenCode. Экспорт в PDF, JPG, PNG.
